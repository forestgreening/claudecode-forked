# 우리, 여기. - 서비스 기획서

> 커플이 함께한 추억을 지도에 기록하고 공유하는 모바일 애플리케이션

---

## 1. 서비스 개요

### 1.1 서비스명
**우리, 여기.**

### 1.2 한 줄 소개
커플이 함께 다녀온 장소를 지도에 기록하고, 추억을 공유하는 위치 기반 커플 다이어리 앱

### 1.3 핵심 가치
- **공유**: 둘만의 프라이빗한 추억 공간
- **시각화**: 지도 위에서 한눈에 보는 우리의 발자취
- **기록**: 사진과 메모로 남기는 소중한 순간들

---

## 2. 문제 정의

### 2.1 핵심 문제
커플들이 함께한 추억을 **공유할 수 있는 전용 공간이 없다.**

### 2.2 현재 상황
- 사진은 각자의 갤러리에 흩어져 있음
- 카카오톡 대화방에 사진을 공유하지만 찾기 어려움
- 기존 커플 앱들은 타임라인(시간순) 중심으로, 장소 기반 회상이 어려움
- "그때 거기서 뭐 했더라?"에 대한 답을 찾기 힘듦

### 2.3 솔루션
**지도 중심 UI**로 장소를 기반으로 추억을 시각화하여, 어떤 장소에서 어떤 추억이 있었는지 직관적으로 확인할 수 있게 함

---

## 3. 타겟 사용자

### 3.1 주요 타겟
추억을 소중히 여기는 **모든 연애/결혼 커플**

### 3.2 페르소나

#### 페르소나 A: 연애 초기 커플 (민지 & 준호, 20대 중반)
- 사귄 지 6개월
- 매주 새로운 데이트 장소 탐방
- 추억을 예쁘게 기록하고 싶어함
- SNS에 올리기보다는 둘만 보고 싶음

#### 페르소나 B: 장기 연애 커플 (수진 & 현우, 20대 후반)
- 사귄 지 3년
- 이미 많은 추억이 있지만 정리가 안 됨
- "우리 그때 어디 갔었지?" 자주 발생
- 여행 다녀온 곳을 한눈에 보고 싶음

#### 페르소나 C: 장거리 연애 커플 (예은 & 태민, 30대 초반)
- 서울-부산 장거리
- 만남이 소중해서 매 순간 기록하고 싶음
- 못 만나는 동안 추억을 보며 버팀

#### 페르소나 D: 신혼 부부 (지영 & 상훈, 30대 초반)
- 결혼 1년 차
- 연애 시절 추억 + 신혼 추억 기록
- 나중에 아이에게 보여주고 싶음

## 4. 경쟁 분석

### 4.1 기존 서비스

| 서비스 | 특징 | 한계 |
|--------|------|------|
| 비트윈 | 타임라인 중심, 채팅 기능 | 지도 기반 시각화 부족 |
| 커플리 | 기념일 관리, 감정 공유 | 장소 기반 기능 미흡 |
| 인스타그램 | 사진 공유 | 프라이빗하지 않음 |
| 구글 포토 | 위치 정보 있음 | 커플 전용 아님 |

### 4.2 차별점
**"지도가 메인 화면"**
- 앱을 열면 바로 지도가 보임
- 우리가 다녀온 모든 장소가 핀으로 표시
- 핀을 탭하면 그 장소의 추억(사진, 메모)이 나옴
- 한눈에 "우리의 발자취"를 확인

## 5. 핵심 기능

### 5.1 MVP 필수 기능 (v1.0)

#### 기능 1: 추억 핀 찍기
- 지도에서 장소 선택 또는 검색
- 현재 위치로 빠르게 핀 생성
- 사진 첨부 (최대 10장)
- 메모/일기 작성
- 날짜 선택 (과거 날짜 가능)
- 카테고리 태그 (카페, 식당, 여행, 산책 등)

#### 기능 2: 커플 연결
- 회원가입 후 초대 코드 생성
- 파트너가 코드 입력하면 연결
- 연결 후 모든 추억 실시간 동기화
- 커플 프로필 설정 (사귄 날짜, 커플 이름 등)

### 5.2 MVP+ 기능 (v1.1)

#### 기능 3: 추억 타임라인
- 시간순으로 추억 목록 보기
- 월별/연도별 필터
- 지도 뷰 ↔ 타임라인 뷰 전환

#### 기능 4: 기념일 등록 및 알림
- 처음 만난 날, 사귄 날 등 기념일 등록
- D-day / D+ 표시
- 기념일 알림 푸시

#### 기능 5: 추억 검색
- 장소명으로 검색
- 날짜로 검색
- 태그로 검색

#### 기능 6: 통계
- 총 방문 장소 수
- 가장 많이 간 장소 TOP 5
- 월별 데이트 횟수
- 지역별 방문 현황

### 5.3 향후 기능 (v2.0+)

#### 기능 7: 데이트 코스 추천
- 현재 위치 기반 주변 추천
- 다른 커플들이 많이 간 곳
- 날씨/시간대별 추천

#### 기능 8: 추억 공유
- 특정 추억을 이미지로 내보내기
- SNS 공유 기능
- 여행 코스 공유

## 6. 사용자 플로우

### 6.1 온보딩 플로우
```
앱 설치 → 회원가입(이메일/소셜) → 프로필 설정
→ 초대 코드 생성 or 입력 → 커플 연결 완료
→ 사귄 날짜 입력 → 메인 지도 화면
```

### 6.2 추억 기록 플로우
```
메인 지도 → + 버튼 탭 → 장소 선택/검색
→ 사진 선택 → 메모 작성 → 카테고리 선택
→ 저장 → 지도에 핀 표시
```

### 6.3 추억 회상 플로우
```
메인 지도 → 핀 탭 → 추억 상세 보기
→ 사진 슬라이드 / 메모 확인 → 수정 or 닫기
```
